Basic Usage Guide¶

The minimum required to fit a model is simply a long-format pandas data-frame with columns for: DV, IVs, and Grouping variables. pymer4 comes with sample data for testing purposes which we’ll utilize here.

Let’s take a quick look at the sample data:

import os
import pandas as pd
import seaborn as sns
from pymer4.models import Lmer
from pymer4.utils import get_resource_path

df = pd.read_csv(os.path.join(get_resource_path(),'sample_data.csv'))
df.head()

For testing purposes we have two different kinds of dependent variables:

  • Continuous (DV)
  • Dichotomous (DV_l)

And two different kinds of independent variables:

  • Continuous (IV1, IV2)
  • Categorical (IV3)

Estimate a simple linear model¶

Fitting a model simply entails initializing it with a model formula and pandas data-frame, and subsequently calling its fit method.

# Initialize model instance
model = Lmer('DV ~ IV2 + (IV2|Group)',data=df)

# Fit it
model.fit()

Inspect random effects¶

We can look at the cluster level parameters easily as they are stored within the model object itself.

model.fixef.head()

Each row here is a unique intercept and slope, which vary because we parameterized our random effects that way above.

We can also plot these values with respect to the population parameters above.

model.plot('IV2',
           plot_ci=True,
           ylabel= 'Predicted DV')

If the model contains more than one predictor variable (unlike above) this plot will visualize a single predictor conditioned on the mean value of every other model term.

Inspect overall fit¶

Using seaborn, it’s trivial to assess model fit using residuals and predictions based on the values that pymer4 stores within its model.data attribute. This attribute its a copy of the data itself with extra appended columns.

sns.regplot(x= 'IV2',
            y= 'residuals',
            data= model.data,
            fit_reg= False
            )

sns.regplot(x= 'fits',
            y= 'DV',
            units= 'Group',
            data= model.data,
            fit_reg= True
            )
